## 一、备案所需材料汇总

### 1.1 法人证书

要求请见"第四章：备案材料示例"

### 1.2 主体负责人资料

#### 第一类：主体负责人是法人

**适用地区：** 部分地区的主体负责人必须是法人

**所需材料：**

1. **法人身份证正反面照片**（照片要求请见"第四章：备案材料示例"）
2. **常用手机号**
   - 未做过其他小程序或公众号备案的手机号
   - 可以不是法人的手机号
   - 可随时接听电话，用于备案期间接收微信平台的核实电话
   - 备案时需填写获取到的短信验证码
3. **应急手机号**
   - 不能跟常用手机号一样
   - 未做过其他小程序或公众号备案的手机号
   - 可以不是法人的手机号
4. **邮箱**
   - 未做过其他小程序或公众号备案的邮箱
   - 可以不是法人的邮箱

#### 第二类：主体负责人不是法人

**适用地区：** 部分地区的主体负责人可以不是法人

**所需材料：**

1. **法人身份证号**
2. **《主体负责人授权书》**
   - 需盖学校公章
   - 法人签名章或法人手写正楷签名
   - 接受签名章，不接受连笔签
   - 模板下载：小程序备案-主体负责人授权书.doc
3. **被授权人身份证正反面照片**（照片要求请见"第四章：备案材料示例"）
4. **常用手机号**
   - 未做过其他小程序或公众号备案的手机号
   - 可随时接听电话，用于备案期间接收微信平台的核实电话
   - 备案时需填写获取到的短信验证码
5. **应急手机号**
   - 不能跟常用手机号一样
   - 未做过其他小程序或公众号备案的手机号
6. **邮箱**
   - 未做过其他小程序或公众号备案的邮箱

**重要提示：** 地区不同，主体负责人要求不同（详见原文档图表）

### 1.3 小程序负责人资料

**所需材料：**

1. **小程序负责人（小程序管理员）身份证正反面照片**（照片要求请见下文）
2. **《小程序负责人授权书》**
   - 需盖学校公章
   - 法人身份证号
   - 法人签名章或法人手写正楷签名
   - 接受签名章，不接受连笔签
   - 模板下载：小程序备案-小程序负责人授权书.doc
3. **常用手机号**
   - 未做过其他小程序或公众号备案的手机号
   - 可随时接听电话，用于备案期间接收微信平台的核实电话
   - 备案时需填写获取到的短信验证码
4. **应急手机号**
   - 不能跟常用手机号一样
   - 未做过其他小程序或公众号备案的手机号
5. **邮箱**
   - 未做过其他小程序或公众号备案的邮箱

**重要提示：** 地区不同，小程序负责人要求不同（详见原文档图表）

## 五、注意事项

1. **手机号要求：** 所有手机号必须未做过其他小程序或公众号备案
2. **邮箱要求：** 邮箱必须未做过其他小程序或公众号备案
3. **授权书要求：** 必须盖学校公章，法人签名（接受签名章，不接受连笔签）
4. **核实电话：** 提交备案后1-2个工作日会接到核实电话，请保持手机畅通
5. **地区差异：** 不同地区对主体负责人和小程序负责人的要求不同，请提前确认
