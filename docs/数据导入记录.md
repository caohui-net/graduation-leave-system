# 数据导入记录

## 原始数据文件

**位置**: `backend/data/`

### 学生数据
- `1-5830名毕业生（含研究生）.csv` (603KB)

### 教职工数据  
- `counselors_processed.csv` (1.3KB) - 辅导员
- `dorm_managers_processed.csv` (3.7KB) - 宿管员

## 导入命令

**脚本位置**: `backend/apps/users/management/commands/import_csv.py`

**导入顺序**（重要）:

```bash
# 1. 辅导员（必须先导入）
python manage.py import_csv --counselors data/counselors_processed.csv

# 2. 班级映射（建立班级-辅导员关系）
python manage.py import_csv --mappings data/class_mappings.csv

# 3. 学生（依赖前两步）
python manage.py import_csv --students data/file5_students_merged_v2.csv
```

**验证命令**:
```bash
python manage.py import_csv --students data/file5_students_merged_v2.csv --dry-run
```
