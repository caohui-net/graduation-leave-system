## 3. UI设计任务范围

### 核心改进点
1. **学生申请页面**：
   - 添加用户信息只读展示区
   - 优化表单样式（卡片式、间距、标签样式）
   - 优化按钮样式（蓝色主色调）

2. **审批列表页面**：
   - 添加Tab切换（全部/待审批/已审批）
   - 优化列表项样式（卡片、状态标签、时间格式）
   - 添加空状态提示

3. **详情页面**：
   - 添加审批流程时间线展示
   - 优化信息分区（申请人信息/离校信息/审批历史/附件）
   - 优化按钮布局（通过/驳回并排）

### 技术约束
- 使用微信小程序原生WXML/WXSS
- 保持现有TS逻辑不变（仅UI层面改动）
- 颜色主题：蓝色主色调（#1890ff或类似）
- 响应式适配：适配不同屏幕宽度
