# 生产数据库备份指南

## 快速备份命令

**在生产服务器上执行**（需要sudo权限）：

```bash
# 方式1: 使用备份脚本（推荐）
sudo /home/caohui/projects/graduation-leave-system/scripts/backup-production-db.sh

# 方式2: 手动备份
sudo mkdir -p /opt/graduation-leave-system/backups/db
cd /opt/graduation-leave-system/production
sudo docker compose -f docker-compose.prod.yml exec -T db pg_dump \
    -U postgres \
    -d graduation_leave \
    --format=custom \
    --blobs \
    --verbose \
    > /opt/graduation-leave-system/backups/db/graduation_leave_$(date +%Y%m%d_%H%M%S).dump
```

## 备份位置

- **目录**: `/opt/graduation-leave-system/backups/db/`
- **命名**: `graduation_leave_prod_YYYYMMDD_HHMMSS.sql`
- **格式**: PostgreSQL custom format (可压缩、可选择性恢复)

## 恢复命令

```bash
# 列出备份文件
ls -lh /opt/graduation-leave-system/backups/db/

# 恢复到新数据库（测试）
sudo docker compose -f docker-compose.prod.yml exec -T db \
    pg_restore -U postgres -d graduation_leave_test \
    < /opt/graduation-leave-system/backups/db/graduation_leave_prod_YYYYMMDD_HHMMSS.sql

# 恢复到生产库（危险！需先停止服务）
sudo docker compose -f docker-compose.prod.yml down
sudo docker compose -f docker-compose.prod.yml up -d db
sudo docker compose -f docker-compose.prod.yml exec -T db \
    dropdb -U postgres graduation_leave
sudo docker compose -f docker-compose.prod.yml exec -T db \
    createdb -U postgres graduation_leave
sudo docker compose -f docker-compose.prod.yml exec -T db \
    pg_restore -U postgres -d graduation_leave \
    < /path/to/backup.sql
sudo docker compose -f docker-compose.prod.yml up -d
```

## 备份策略

- **手动备份**: 重大操作前（数据迁移、schema变更、版本升级）
- **自动备份**: 每日凌晨2点（通过cron）
- **保留周期**: 7天（自动清理旧备份）

## Cron定时备份（可选）

```bash
# 在生产服务器上添加cron任务
sudo crontab -e

# 添加以下行（每日凌晨2点备份）
0 2 * * * /home/caohui/projects/graduation-leave-system/scripts/backup-production-db.sh >> /opt/graduation-leave-system/backup-cron.log 2>&1
```

## 验证备份

```bash
# 检查备份文件大小
ls -lh /opt/graduation-leave-system/backups/db/

# 测试备份完整性
sudo docker compose -f docker-compose.prod.yml exec -T db \
    pg_restore --list /tmp/graduation_leave_prod_YYYYMMDD_HHMMSS.sql | head -20

# 查看备份日志
cat /opt/graduation-leave-system/backup.log
```
