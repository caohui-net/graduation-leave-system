## 二、技术实现

### 数据库变更
```sql
-- 新增字段
ALTER TABLE applications_application 
  ADD COLUMN application_type VARCHAR(20) DEFAULT 'leave_school',
  ADD COLUMN stay_start_date DATE NULL,
  ADD COLUMN stay_end_date DATE NULL,
  ADD COLUMN stay_reason VARCHAR(30) NULL;
```

### 审批流程路由
- **离校申请**: 宿管员 → 辅导员
- **留校申请**: 辅导员（跳过宿管员）
- **请假申请**: 暂不开放

### Feature Flag控制
```python
FEATURE_FLAGS = {
    'stay_school_approval': os.getenv('ENABLE_STAY_SCHOOL', 'false') == 'true',
    'leave_request_approval': os.getenv('ENABLE_LEAVE_REQUEST', 'false') == 'true',
}
```
