# CSV字段映射

## 学生数据映射

### 毕业生CSV → 数据库

| 数据库字段 | CSV列名 | 处理方式 | 示例 |
|-----------|---------|---------|------|
| user_id | （姓名匹配） | 通过姓名查询补全 | 2025150140636 |
| name | 学生姓名 | 直接映射 | 雷力 |
| class_id | 班级 | 直接映射 | 电信(专升本)202403 |
| campus | 校区 | 直接映射 | 北区 |
| building | 楼栋名称 | 直接映射 | 荷园2栋 |
| room_number | 寝室号 | 直接映射 | 206 |
| gender | 性别 | 直接映射 | 女 |
| major | 专业 | 直接映射 | 电子信息工程 |
| level | 层次 | 直接映射 | 专升本 |

### 固定值字段

| 字段 | 固定值 | 说明 |
|------|--------|------|
| role | "student" | 所有学生统一角色 |
| is_graduating | true | 从毕业生名单导入 |
| graduation_year | 2026 | 根据年级(2024)推算 |
| active | true | 默认激活状态 |
