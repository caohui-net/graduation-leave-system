## 管理员操作指南

### 1. 登录系统

**操作步骤：**

1. 打开系统登录页面
2. 输入工号（如：D001）
3. 输入密码（初始密码为工号）
4. 点击【登录】按钮

**界面说明：**

![管理员登录界面](screenshots/admin-login.png)

*图15：管理员登录界面*

### 2. 查询所有申请

**操作步骤：**

1. 登录后进入【申请管理】页面
2. 查看所有学生的离校申请列表
3. 可按条件筛选：
   - 状态：待审批/已通过/已驳回/全部
   - 学院：计算机学院/软件学院/全部
   - 时间范围：选择日期区间

**界面说明：**

![管理员申请列表](screenshots/admin-application-list.png)

*图16：管理员申请列表界面*

**列表信息：**
- 学生姓名
- 学号
- 班级
- 学院
- 宿舍楼栋
- 申请时间
- 当前状态
- 操作按钮

**筛选功能：**
- 【待审批】：显示所有待审批的申请
- 【审批中】：显示正在审批流程中的申请
- 【已完成】：显示已通过或已驳回的申请
- 【全部】：显示所有申请

### 3. 查看申请详情

**操作步骤：**

1. 在申请列表中点击某个申请
2. 进入申请详情页面
3. 查看完整的审批记录

**界面说明：**

![管理员申请详情](screenshots/admin-application-detail.png)

*图17：管理员申请详情页面*

**详情页面包含：**
- **基本信息：** 申请编号、学生信息、提交时间、当前状态
- **申请内容：** 离校原因、计划离校日期
- **完整审批记录：**
  - 宿管员审批（审批人、时间、结果、意见）
  - 辅导员审批（审批人、时间、结果、意见）
- **操作按钮：** 返回列表、导出记录

### 4. 查询审批记录

**操作步骤：**

1. 点击【审批记录】菜单
2. 查看所有审批记录
3. 可按条件筛选：
   - 审批人：宿管员/辅导员
   - 审批结果：通过/驳回/全部
   - 时间范围：选择日期区间
   - 楼栋/学院：按组织筛选

**界面说明：**

![管理员审批记录](screenshots/admin-approval-history.png)

*图18：管理员审批记录页面*

**记录信息：**
- 申请编号
- 学生姓名
- 审批人姓名
- 审批步骤（宿管员/辅导员）
- 审批结果
- 审批时间
- 审批意见

### 5. 导出统计报表

**操作步骤：**

1. 在申请列表或审批记录页面
2. 设置筛选条件
3. 点击【导出报表】按钮
4. 选择导出格式（Excel/CSV）
5. 下载统计报表

**报表内容：**
- 申请统计数据
- 审批通过率
- 各学院/楼栋分布
- 审批时效统计

### 6. 权限说明

**管理员权限：**
- ✓ 查看所有学生的离校申请
- ✓ 查看所有审批记录
- ✓ 导出统计报表
- ✓ 查看系统运行状态
- ✗ 不能直接审批申请（审批权限在宿管员和辅导员）
- ✗ 不能修改或删除申请

**注意事项：**
- 管理员主要用于数据查询和存档
- 不参与具体的审批流程
- 可以查看所有数据用于统计分析

## 常见问题（管理员）

**Q1：如何查看某个学院的所有申请？**

A1：在申请列表页面，使用学院筛选条件，选择目标学院即可。

**Q2：如何统计审批通过率？**

A2：使用导出报表功能，设置时间范围，导出Excel文件后可查看统计数据。

**Q3：管理员可以直接审批吗？**

A3：不可以。管理员只有查询权限，审批由宿管员和辅导员完成。

**Q4：如何查看某个时间段的所有申请？**

A4：使用时间筛选功能，设置开始日期和结束日期即可。

**Q5：导出的报表包含哪些内容？**

A5：包含申请基本信息、审批记录、统计数据（通过率、平均审批时长等）。
