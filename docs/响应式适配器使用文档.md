# 响应式适配器使用文档

## 简介

响应式适配器是一个轻量级的移动端自动适配模块，可自动检测设备类型和屏幕尺寸，动态调整UI。

**特性：**
- 📱 自动设备检测（手机/平板/桌面）
- 📏 屏幕尺寸分类（xs/sm/md/lg/xl）
- 🎨 自动添加CSS类名
- 📡 事件监听（屏幕变化/设备变化）
- 🔧 可配置的断点
- 🚀 零依赖，<5KB

---

## 快速开始

### 1. 引入模块

```html
<script src="js/responsive-adapter.js"></script>
```

### 2. 初始化

```javascript
// 自动模式（推荐）
const adapter = ResponsiveAdapter.create();

// 自定义配置
const adapter = ResponsiveAdapter.create({
  breakpoints: {
    xs: 360,
    sm: 480,
    md: 768,
    lg: 1024,
    xl: 1280
  },
  classPrefix: 'screen-',      // CSS类名前缀
  enableAutoClass: true,        // 自动添加CSS类
  enableResize: true,           // 监听窗口变化
  debounceDelay: 150           // 防抖延迟(ms)
});
```

### 3. 使用API

```javascript
// 判断设备类型
if (adapter.isMobile()) {
  console.log('手机端');
}

// 获取屏幕尺寸分类
const size = adapter.getScreenSize(); // 'xs'|'sm'|'md'|'lg'|'xl'

// 获取设备信息
const info = adapter.getDeviceInfo();
console.log(info);
// {
//   screen: 'sm',
//   device: 'mobile',
//   width: 375,
//   height: 667,
//   os: 'iOS',
//   isTouch: true,
//   pixelRatio: 2,
//   isRetina: true
// }
```

## API 参考

### 设备判断

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `isMobile()` | Boolean | 是否为手机 (≤768px) |
| `isTablet()` | Boolean | 是否为平板 (768-1024px) |
| `isDesktop()` | Boolean | 是否为桌面 (>1024px) |
| `isTouchDevice()` | Boolean | 是否为触摸设备 |
| `isIOS()` | Boolean | 是否为iOS系统 |
| `isAndroid()` | Boolean | 是否为Android系统 |
| `isRetina()` | Boolean | 是否为高清屏 |

### 尺寸获取

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `getWidth()` | Number | 屏幕宽度(px) |
| `getHeight()` | Number | 屏幕高度(px) |
| `getScreenSize()` | String | 尺寸分类 ('xs'/'sm'/'md'/'lg'/'xl') |
| `getDeviceType()` | String | 设备类型 ('mobile'/'tablet'/'desktop') |
| `getOS()` | String | 操作系统 |
| `getPixelRatio()` | Number | 设备像素比 |

### 事件监听

```javascript
// 监听屏幕尺寸变化
adapter.on('screenChange', (data) => {
  console.log(`从 ${data.from} 变为 ${data.to}`);
  console.log(`当前宽度: ${data.width}px`);
});

// 监听设备类型变化
adapter.on('deviceChange', (data) => {
  console.log(`从 ${data.from} 变为 ${data.to}`);
});

// 移除监听
adapter.off('screenChange', callback);
```

## 使用示例

### 示例1：基础集成

```html
<!DOCTYPE html>
<html>
<head>
  <script src="js/responsive-adapter.js"></script>
  <style>
    /* 基于自动添加的类名 */
    .screen-xs .content { font-size: 12px; }
    .screen-sm .content { font-size: 14px; }
    .screen-md .content { font-size: 16px; }
    .screen-lg .content { font-size: 18px; }
    .screen-xl .content { font-size: 20px; }

    .device-mobile .sidebar { display: none; }
    .device-desktop .sidebar { display: block; }
  </style>
</head>
<body>
  <script>
    const adapter = ResponsiveAdapter.create();
    console.log('当前设备:', adapter.getDeviceInfo());
  </script>
</body>
</html>
```

### 示例2：动态加载组件

```javascript
const adapter = ResponsiveAdapter.create();

// 根据设备类型加载不同组件
if (adapter.isMobile()) {
  loadMobileNavigation();
} else {
  loadDesktopNavigation();
}

// 监听屏幕变化，动态切换
adapter.on('deviceChange', (data) => {
  if (data.to === 'mobile') {
    switchToMobileLayout();
  } else {
    switchToDesktopLayout();
  }
});
```

### 示例3：快速判断（无需初始化）

```javascript
// 直接使用快速API
if (ResponsiveAdapter.isMobile()) {
  console.log('手机端');
}

const width = ResponsiveAdapter.getWidth();
console.log(`屏幕宽度: ${width}px`);
```

### 示例4：自定义断点

```javascript
// 针对大屏项目调整断点
const adapter = ResponsiveAdapter.create({
  breakpoints: {
    xs: 320,
    sm: 600,
    md: 960,
    lg: 1280,
    xl: 1920
  }
});
```

## 与当前项目集成

### 方式1：在index.html引入

```html
<head>
  <meta charset="UTF-8">
  <script src="js/responsive-adapter.js"></script>
  <script src="js/api.js?v=20260612"></script>
</head>
```

### 方式2：在现有代码中使用

```javascript
// 在api.js或index.html的<script>中
const adapter = ResponsiveAdapter.create();

// 优化筛选按钮显示
function updateFilterTabs() {
  const tabs = document.querySelectorAll('.filter-tab');
  
  if (adapter.getScreenSize() === 'xs') {
    // 超小屏：只显示图标
    tabs.forEach(tab => {
      tab.textContent = tab.textContent.split(' ')[0]; // 只保留emoji
    });
  } else {
    // 其他屏幕：图标+文字
    // 保持原样
  }
}

// 监听变化
adapter.on('screenChange', updateFilterTabs);
```
