## 1. 需求说明

**用户要求：** "第一审批人学校要求是宿管员，第二审批人是辅导员"

### 当前审批流程

```
学生提交申请 → 辅导员审批 → 院长/学工部审批 → 完成
```

### 目标审批流程

```
学生提交申请 → 宿管员审批 → 辅导员审批 → 完成
```
