## 根本原因分析

### 数据源差异

```
Excel文件（File1+File2） → merge → File5（5946人）
                                     ↓
                          Claude分析（116人无楼栋）

用户系统数据库 → 导出 → 1-无入住信息.xls（134人）
```

**核心问题:** Excel文件与用户系统数据库不同步

### 差异来源

1. **时间维度**
   - Excel文件：某个时间点的静态导出
   - 用户系统：实时数据库，可能包含更新数据

2. **数据范围**
   - Excel文件：5946名学生（File1 5830 + File2独有 116）
   - 用户系统：可能包含更多学生（退学、转学、补录等）

3. **数据质量**
   - 19名学生：用户系统有但Excel无
   - 1名学生：Excel有但用户系统无
