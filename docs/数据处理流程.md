# 数据处理流程

## 流程概览

```
原始毕业生CSV → 学号匹配 → 合并处理 → 最终导入CSV → 数据库
```

## 详细步骤

### 步骤1: 学号匹配

**输入**:
- `1-5830名毕业生（含研究生）.csv` - 含姓名、宿舍，缺学号

**处理**:
- 通过姓名从其他数据源匹配学号
- 生成 `missing_student_no_filled.csv` (31KB) - 匹配结果

**输出**:
- 补全学号的学生记录

### 步骤2: 数据合并

**输入**:
- 原始毕业生CSV（宿舍信息）
- 学号匹配结果

**处理**:
- 合并字段: 学号 + 姓名 + 班级 + 宿舍信息
- 添加固定字段: is_graduating=true, graduation_year=2026

**输出**:
- `file5_students_merged_v2.csv` (788KB) - 最终导入文件

### 步骤3: 辅导员处理

**输入**:
- `2026年学院辅导员信息统计表.csv`
- `2026年社区辅导员信息统计表.csv`

**处理**:
- 合并学院+社区辅导员
- 标准化格式: employee_id, name

**输出**:
- `counselors_processed.csv` (1.3KB)

## 数据关系

```
辅导员 ──1:N── 班级映射 ──1:N── 学生
   ↓                          ↓
counselors_processed    file5_students_merged_v2
```

**约束**: 学生的class_id必须在班级映射中存在
