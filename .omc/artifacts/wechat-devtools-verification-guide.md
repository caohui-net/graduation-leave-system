## 常见问题

### Q1: 导入时提示"不是有效的小程序项目"
**原因：** 缺少 `app.json` 或格式错误  
**解决：** 检查 `miniprogram/app.json` 是否存在且格式正确

### Q2: 编译时提示"找不到模块"
**原因：** TypeScript路径配置或依赖缺失  
**解决：** 检查 `tsconfig.json` 和 `package.json`

### Q3: 真机预览时网络请求失败
**原因：** localhost不可访问  
**解决：** 使用内网IP或配置内网穿透

### Q4: wx.request不是函数
**原因：** 使用了Web API (fetch) 而非小程序API  
**解决：** 修改API client使用 `wx.request`
