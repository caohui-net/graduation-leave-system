```
POST http://218.75.196.218:7787/api/sso/qingganlian/mobile/login
```


## 五、技术实现说明

### 5.1 认证流程

**移动端：**
1. 用户在青橄榄移动应用登录，获取saas_wap_token
2. 前端调用我方登录接口，传递token
3. 我方后端调用青橄榄API验证token并获取用户信息
4. 创建/匹配本地用户，返回本地JWT
5. 后续请求使用本地JWT认证

**管理端：**
1. 用户在一站式管理平台点击应用
2. 平台跳转到我方系统，URL携带Authorization参数
3. 前端提取token，调用我方登录接口
4. 我方后端调用青橄榄API验证token
5. 创建/匹配管理员用户，返回本地JWT

### 5.2 用户映射

- 建立青橄榄user_code/username → 本地用户映射表
- 首次登录时创建本地用户记录
- 后续登录直接从映射表查询，减少API调用

### 5.3 安全措施

- 所有API请求携带签名验证
- 青橄榄token仅用于一次性交换
- 本地JWT独立管理，有效期1小时
- appKey/appSecret存储在服务器环境变量
