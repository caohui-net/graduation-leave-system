## Staging部署步骤

### 1. 数据库迁移（dry-run）
```bash
cd backend
source venv/bin/activate
python manage.py migrate --settings=config.staging --dry-run
```

### 2. 执行迁移
```bash
python manage.py migrate --settings=config.staging
```

### 3. 验证字段
```sql
\d applications;
-- 确认字段存在：
-- application_type, stay_start_date, stay_end_date, stay_reason
```

### 4. 部署代码
```bash
# 后端
git pull origin main
systemctl restart graduation-backend-staging

# 前端
cd frontend
npm run build
# 部署dist/到staging服务器
```

### 5. Feature Flag测试
```bash
# 关闭状态验证
ENABLE_STAY_SCHOOL=false
# 测试：提交留校申请应返回403

# 开启状态验证
ENABLE_STAY_SCHOOL=true
# 测试：提交留校申请应成功
```
