## 三、HTTPS/SSL证书问题的最终方案

**决策：** 支持HTTP和HTTPS双模式，默认HTTP

**实施方案：**
1. **默认配置：** HTTP（内网部署，配合IP白名单+API Key）
2. **可选配置：** HTTPS（有公网域名和SSL证书时）
3. **文档说明：** 明确两种模式的适用场景和安全措施
4. **部署支持：** Docker Compose默认HTTP，提供Nginx HTTPS配置模板

**安全平衡：**
- 内网环境：HTTP + IP白名单 + API Key（风险可控）
- 公网环境：HTTPS + API Key + IP白名单（最高安全）
