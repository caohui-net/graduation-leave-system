## 系统简介

### 系统概述

毕业生离校申请审批系统是为毕业生办理离校手续而开发的在线审批平台。系统支持学生在线提交离校申请，经宿管员和辅导员两级审批后，学生即可完成离校手续。

### 审批流程

```
学生提交申请 → 宿管员审批 → 辅导员审批 → 审批完成
```

**流程说明：**
1. **学生提交：** 填写离校申请表，说明离校原因和计划离校日期
2. **宿管员审批：** 确认学生宿舍物品清理完毕，无欠费等问题
3. **辅导员审批：** 确认学生档案、学籍等事项办理完毕
4. **审批完成：** 学生可查看审批结果，完成离校

### 系统角色

- **学生：** 提交离校申请，查看审批进度
- **宿管员：** 审批本楼栋学生的离校申请
- **辅导员：** 审批本学院学生的离校申请
- **学工部：** 查看所有审批记录（存档）

## 学生操作指南

### 1. 登录系统

**操作步骤：**

1. 打开系统登录页面
2. 输入学号（如：2020001）
3. 输入密码（初始密码为学号）
4. 点击【登录】按钮

**界面说明：**

![登录界面](screenshots/student-login.png)

*图1：学生登录界面*

- **学号输入框：** 输入10位学号
- **密码输入框：** 输入密码（初始密码为学号）
- **登录按钮：** 点击进入系统

**注意事项：**
- 首次登录后建议修改密码
- 忘记密码请联系辅导员重置

### 2. 提交离校申请

**操作步骤：**

1. 登录后进入【首页】
2. 点击【提交离校申请】按钮
3. 填写申请表单：
   - 离校原因：说明离校原因（如"已完成毕业手续"）
   - 计划离校日期：选择预计离校日期
4. 确认信息无误后，点击【提交】按钮

**界面说明：**

![提交申请界面](screenshots/student-submit-application.png)

*图2：提交离校申请界面*

**表单字段：**
- **离校原因：** 必填，文字说明，建议20-100字
- **计划离校日期：** 必填，日期选择，不能早于当前日期

**提交前检查：**
- ✓ 确认宿舍物品已清理
- ✓ 确认无宿舍欠费
- ✓ 确认图书馆无欠书
- ✓ 确认档案已办理

**提交后：**
- 系统显示"提交成功"提示
- 自动进入申请详情页面
- 可以查看审批进度

### 3. 查看申请状态

**操作步骤：**

1. 点击【我的申请】
2. 查看申请列表
3. 点击具体申请可查看详情

**界面说明：**

![申请列表界面](screenshots/student-application-list.png)

*图3：我的申请列表*

**申请状态说明：**

| 状态 | 说明 | 后续操作 |
|------|------|----------|
| 待宿管员审批 | 宿管员正在审核中 | 等待审批 |
| 待辅导员审批 | 宿管员已通过，辅导员审核中 | 等待审批 |
| 已通过 | 所有审批通过，可以离校 | 无 |
| 已驳回 | 审批未通过 | 查看驳回原因 |

### 4. 查看申请详情

**操作步骤：**

1. 在申请列表中点击某个申请
2. 进入申请详情页面
3. 查看审批记录和审批意见

**界面说明：**

![申请详情界面](screenshots/student-application-detail.png)

*图4：申请详情页面*

**详情页面包含：**
- **基本信息：** 申请编号、提交时间、当前状态
- **申请内容：** 离校原因、计划离校日期
- **审批记录：** 各级审批人、审批时间、审批意见
- **操作按钮：** 返回列表

**审批记录说明：**

![审批记录](screenshots/student-approval-history.png)

*图5：审批记录详情*

- **宿管员审批：** 显示宿管员姓名、审批时间、审批结果、审批意见
- **辅导员审批：** 显示辅导员姓名、审批时间、审批结果、审批意见

### 5. 处理驳回申请

**如果申请被驳回：**

1. 查看驳回原因
2. 按照驳回意见完成相关事项
3. 重新提交申请

**注意事项：**
- 已驳回的申请无法修改，需要重新提交
- 重新提交前请确认驳回原因已解决
- 同一时间只能有一个待审批或已通过的申请

## 宿管员操作指南

### 1. 登录系统

**操作步骤：**

1. 打开系统登录页面
2. 输入工号（如：M001）
3. 输入密码（初始密码为工号）
4. 点击【登录】按钮

**界面说明：**

![宿管员登录界面](screenshots/manager-login.png)

*图6：宿管员登录界面*

### 2. 查看待审批列表

**操作步骤：**

1. 登录后进入【待审批】页面
2. 查看本楼栋学生的离校申请列表
3. 可按状态筛选：待审批、已审批、全部

**界面说明：**

![宿管员待审批列表](screenshots/manager-approval-list.png)

*图7：宿管员待审批列表*

**列表信息：**
- 学生姓名
- 学号
- 班级
- 宿舍楼栋/房间
- 申请时间
- 当前状态
- 操作按钮

**筛选功能：**
- 【待审批】：只显示等待本人审批的申请
- 【已审批】：显示已处理的申请
- 【全部】：显示所有申请

### 3. 审批申请

**操作步骤：**

1. 在待审批列表中点击【审批】按钮
2. 查看申请详情
3. 核实学生宿舍清理情况
4. 选择审批结果：通过或驳回
5. 填写审批意见
6. 点击【提交审批】

**界面说明：**

![宿管员审批界面](screenshots/manager-approve.png)

*图8：宿管员审批界面*

**审批要点：**

**通过条件：**
- ✓ 宿舍物品已清理干净
- ✓ 床铺、桌椅无损坏
- ✓ 无水电费欠费
- ✓ 钥匙已归还

**驳回情况：**
- ✗ 宿舍物品未清理
- ✗ 有损坏赔偿未处理
- ✗ 有欠费未缴纳
- ✗ 钥匙未归还

**审批意见填写：**
- 通过时：简要说明（如"宿舍清理完毕，同意离校"）
- 驳回时：详细说明原因（如"宿舍桌椅损坏未赔偿"）

### 4. 多宿管员协同审批

**说明：** 同一楼栋如果有多个宿管员，任意一个宿管员审批通过后，其他宿管员会看到"已由XXX完成审批"的提示。

**操作步骤：**

1. 如果看到某申请显示"已由其他宿管员审批"
2. 可以查看该宿管员的审批意见
3. 无需重复审批

**界面说明：**

![多宿管员提示](screenshots/manager-multi-approval.png)

*图9：已由其他宿管员完成审批提示*

### 5. 查看审批记录

**操作步骤：**

1. 点击【审批记录】
2. 查看已处理的申请
3. 可筛选时间范围

**界面说明：**

![审批记录](screenshots/manager-history.png)

*图10：审批记录页面*

## 常见问题

### 学生常见问题

**Q1：忘记密码怎么办？**

A1：请联系辅导员重置密码。初始密码为学号，建议首次登录后修改。

**Q2：提交申请后可以修改吗？**

A2：已提交的申请无法修改。如需修改，请联系宿管员驳回申请，然后重新提交。

**Q3：为什么无法提交申请？**

A3：可能原因：
- 已有待审批或已通过的申请（同一时间只能有一个）
- 宿舍清退未完成
- 网络连接问题

**Q4：申请被驳回后怎么办？**

A4：查看驳回原因，按要求完成相关事项后，重新提交申请。

**Q5：审批需要多长时间？**

A5：通常情况下，宿管员审批1-2个工作日，辅导员审批1-2个工作日。

### 宿管员常见问题

**Q1：如何判断学生宿舍是否清理完毕？**

A1：现场检查宿舍，确认：
- 个人物品已搬走
- 床铺整洁
- 桌椅无损坏
- 地面清洁
- 钥匙已归还

**Q2：同楼栋有多个宿管员怎么办？**

A2：任意一个宿管员审批即可，其他宿管员会看到"已由XXX完成审批"提示。

**Q3：误操作通过或驳回怎么办？**

A3：审批一旦提交无法撤销。如误操作，请联系技术支持处理。

### 辅导员常见问题

**Q1：如何查看学生档案办理情况？**

A1：可在教务系统中查询学生档案状态。

**Q2：驳回申请后学生会收到通知吗？**

A2：是的，系统会通知学生查看驳回原因。

**Q3：可以批量审批吗？**

A3：当前版本暂不支持批量审批，需逐个审批。
